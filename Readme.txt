1. Создаем папку для хранения нашего скрипта через mkdir project
2. Затем через команду touch cldir.sh создаем исполняемый файл шде будет написан наш скрипт
3. Открываем файл через Nautilus и пишем сначала функцию которая будет проверять существует ли директория
4. Потом пишем функцию которая будет очищать директорию от файлов с определенным расширением
5. В случае если директория не существует выводим ошибку
